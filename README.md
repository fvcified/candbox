# Candbox

A personal sandbox for learning and experimenting with C programming.

## Topics Covered

* Variables & Data Types
* Operators
* Conditionals
* Loops
* Arrays & Strings
* Functions
* Enums & Structs
* Pointers
* Dynamic Memory (`malloc`, `free`)
* File Handling
* Modular Programming (`.c` and `.h` files)

## Project Structure

```text
candbox/
├─ .gitignore
├─ README.md
├─ demos.c
├─ demos.h
├─ file_utils.h
├─ file_utils.c
├─ main.c
├─ user.c
├─ user.h
└─ user.txt
```

## Compile and Run

Compile all source files:

```bash
gcc *.c -o candbox
```

Run the program:

```bash
./candbox
```
